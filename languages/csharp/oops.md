## Classes and Struct
`struct` &rarr; value type, saved on stack, no OOPS, and used for smaller stuff

`class` &rarr; reference type, saved on heap, OOPS, and used for complex stuff

```c#
class/struct Point {
    public int X;
    public int Y;
}

Point p1 = new Point{X=1, Y=2};
p1.X = 3
```
