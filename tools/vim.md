## Basics
Vim is a Text Editor that is completely keyboard based. A list of all the commands can be found [here](https://vimsheet.com/). some of the most important are as follows:

## Modes
`Normal mode` &rarr; navigation & commands

`Insert mode` &rarr; typing text

`Visual mode` &rarr; selecting text

`Command mode` &rarr; saving, quitting, settings

## Save and Quit
`:w` &rarr; save

`:q` &rarr; quit

`:wq` &rarr; save + quit

`:q!` &rarr; quit without saving

## File Navigation
`w` &rarr; jump to next word <br>
`b` &rarr; jump back a word <br>
`e` &rarr; end of word foward
`ge` &rarr; end of word backward

`0` &rarr; start of line <br>
`$` &rarr; end of line

`gg` &rarr; top of file <br>
`G` &rarr; bottom of file

## Delete
`x` &rarr; delete character <br>
`dd` &rarr; delete line <br>
`dw` &rarr; delete word forward <br>
`d3b` &rarr; delete 3 words backward <br>
`d$` &rarr; delete to end of line <br>

## Copy/Paste
`yy` &rarr;copy line <br>
`yw` &rarr;copy word <br>

`p` &rarr; paste after <br>
`P` &rarr; paste before

## Search
`/word` &rarr; in Normal Mode <br>
`n` &rarr; next match <br>
`N` &rarr; previous match

## Other
<b>Movement</b> &rarr; `h` (left), `j` (down), `k` (up), `l` (right)

<b>Undo/Redo</b> &rarr; `u` (undo), `ctrl+r` (redo)

<b>Go to Line</b> &rarr; `:21`

<b>Move Line</b> &rarr; `:m 20`
